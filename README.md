# Digital_Humidity_Controller
# 湿度传感器数字控制器 (Digital Humidity Controller)

## 项目背景
本项目是一个基于 Verilog HDL 开发的硬件控制器，专门用于处理传感器采集的湿度数据。该项目将我之前的“湿敏材料研究”与“数字电路设计”结合，实现了从底层材料特性到顶层控制逻辑的垂直整合。

## 功能描述
- **自动控制**：当湿度超过 80% 时，自动开启风扇 (`fan_on`)。
- **迟滞保护**：湿度降至 40% 以下才关闭，防止风扇频繁启停。
- **高湿报警**：湿度超过 95% 时触发报警信号 (`alarm`)。

## 设计细节
- 采用 **FSM (有限状态机)** 设计。
- 经过同步时序逻辑处理，可直接综合到 FPGA 或 ASIC。
